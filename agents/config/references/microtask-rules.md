# Microtask Rules

## Principle

Microtasks exist to protect momentum, not to create bureaucracy.

## Good microtasks

- follow up in 3 days because lead asked for time to check internal availability
- create weekly warm-lead review because many qualified leads are stalling with no callback
- reactivation in 30 days because the buyer said the project is for next month
- ask marketing agent for revised ad angle because sales is receiving low-fit inquiries

## Bad microtasks

- generic “check in later” with no timing or reason
- daily reminders on leads who asked not to be contacted
- recurring follow-ups with no change in message or purpose
- tasks that only restate the conversation log

## Trigger types

### Immediate
Use when the next action should happen now.

### Date-based
Use when the buyer named a time window.

### Event-based
Use when something must happen after a milestone, such as a campaign launch or meeting.

### Recurring
Use only when the work is truly cyclical and valuable.

## Task naming style

Use short names with verbs:
- retomar lead oficina centro
- revisar qualidade campanha manutencao
- cobrar insumo do marketing
- reativar oportunidade abril
